Java MongoDB Connection (Using Eclipse and Maven) This document provides a step-by-step lab guide for connecting Java with MongoDB using Eclipse and Maven. Step 1: Prerequisites - Install MongoDB and ensure it is running on localhost:27017 - Install Eclipse IDE for Java Developers - Install Java JDK 8 or higher Step 2: Create a Maven Project 1. Open Eclipse File New Maven Project 2. Select 'Create a simple project (skip archetype selection)' 3. Enter the following: Group ID: com.example Artifact ID: JavaMongo 4. Click Finish. Step 3: Add MongoDB Dependency Open `pom.xml` and add the following dependency inside `<dependencies>`: <dependency> <groupId>org.mongodb</groupId> <artifactId>mongodb-driver-sync</artifactId> <version>4.10.2</version> </dependency>
Step 4: Create Package and Java Class 1. Go to src/main/java 2. Create a new package named: mongoJava 3. Inside it, create a new class: javamongo.java Step 5: Add the Following Code First program Second Program package mongoJava; import com.mongodb.client.MongoClient; import com.mongodb.client.MongoClients; import com.mongodb.client.MongoCollection; import com.mongodb.client.MongoCursor; import com.mongodb.client.MongoDatabase; import org.bson.Document; public class javamongo { public static void main(String[] args) { try (MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017")) { System.out.println("Successfully created MongoDB connection"); MongoDatabase db = mongoClient.getDatabase("mongodbjava"); System.out.println(" Connected to database: " + db.getName()); System.out.println(" Databases present in MongoDB:"); MongoCursor<String> dbsCursor = mongoClient.listDatabaseNames().iterator(); while (dbsCursor.hasNext()) {
System.out.println(" - " + dbsCursor.next()); } MongoCollection<Document> collection = db.getCollection("javaprogram"); Document doc = new Document("name", "hello world"); collection.insertOne(doc); System.out.println(" Document inserted successfully"); MongoCursor<Document> cursor = collection.find().iterator(); System.out.println("\n Documents in 'javaprogram' collection:"); while (cursor.hasNext()) { System.out.println(cursor.next().toJson()); } } catch (Exception e) { e.printStackTrace(); } } } Step 6: Run the Program 1. Start MongoDB service (default port 27017) 2. In Eclipse, right-click the file Run As Java Application 3. Observe the console output: - Connection success message - Database list - Insertion success - Documents displayed
